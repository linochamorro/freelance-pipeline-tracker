# Product

## Register

product

## Users

Freelancers and independent professionals who manage multiple client opportunities simultaneously. They use this tool during daily workflow — between client calls, proposal writing, and project delivery — to keep track of where each prospect stands in the pipeline. They are comfortable with web apps and expect fast, keyboard-accessible interactions.

## Product Purpose

A pipeline tracker that helps freelancers move opportunities from first contact to completed project and payment. It surfaces what needs attention (stale opportunities, conversion metrics) without getting in the way of the actual work.

## Brand Personality

Competent, direct, unpretentious. Three words: **sharp, calm, dependable**.

## Anti-references

- Over-designed SaaS dashboards with gradient charts, heavy illustrations, and decorative animations.
- CRM-style clutter with too many fields and panels per screen.
- Dark mode as default "because tools look cool dark." The surface is light, clean, and readable.
- Playful or whimsical branding. This is a work tool.

## Design Principles

1. **The tool disappears.** Every screen, component, and interaction should feel like it was already there. The user's task — not the UI — is the focus.
2. **State over decoration.** Color, motion, and emphasis exist to communicate status (stale, active, completed, error), not to decorate.
3. **Progressive disclosure.** Start with the essential. Reveal secondary controls, filters, and metadata only when the user needs them.
4. **Consistency is trust.** The same pattern works the same way everywhere — same button hierarchy, same form vocabulary, same icon behavior.
5. **Accessibility is not a mode.** Keyboard navigation, screen reader support, and sufficient contrast are built into every component from the start.

## Accessibility & Inclusion

- WCAG 2.2 AA minimum.
- Full keyboard navigation for all interactive surfaces, especially the Kanban board.
- Respect prefers-reduced-motion.
- Semantic HTML and ARIA throughout.
- Body text contrast ≥ 4.5:1 against all surfaces.
